On this page, you as a manager are able to manage and register company users.

Manage existing user

Here you can manage user, which is aready registered in your system.

Use email in format e.g. example@example.com.
Use first name in format e.g. John, with min. of 3 characters and first capital letter.
Use middle name in format e.g. Peter, with min. of 3 characters and first capital letter.
Use last name in format e.g. Smith, with min. of 3 characters and first capital letter.
Use phone number in format e.g. +421111222333, with min. of 6 characters and recomended country prefix.
One of the options has to be selected.

Register user

Here you can register a new user.

Use email in format e.g. example@example.com.
Use first name in format e.g. John, with min. of 3 characters and first capital letter.
Use middle name in format e.g. Peter, with min. of 3 characters and first capital letter.
Use last name in format e.g. Smith, with min. of 3 characters and first capital letter.
Please generate new password.
One of the options has to be selected.
Use phone number in format e.g. +421111222333, with min. of 6 characters and recomended country prefix.
